# awl editor

A fast native editor with mg/Emacs keybindings.

## Motions
- C-f forward, C-b backward
- C-n next line, C-p previous line
- M-f / M-b word motion
- C-a / C-e line start/end
- M-< / M-> buffer start/end

## Editing
Type to insert. C-k kills a line, C-y yanks it back.
Save with C-x C-s, quit with C-x C-c.
