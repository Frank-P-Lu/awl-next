# awl editor

A calm, native editor for writing — and light editing.
It opens in Tawny; press ⌘T to wander the other worlds.

## Find your way
- ⌘P    command palette — every command, by name
- ⌘O    go to a file (fuzzy)
- ⌘⇧P   switch project
- ⌘F    find in this file
- Hold ⌘   peek the shortcuts you haven't learned yet

## Write
Type to insert. ⌘Z undo, ⌘X / ⌘C / ⌘V cut, copy, paste.
⌘B bold, ⌘E inline code — the rest of the formatting toggles live in the palette (⌘P).

## Look
- ⌘T          switch theme — fourteen worlds, each with its own type
- ⌘P → Caret style…   solid block, or a glyph that morphs as it moves
- ⌘+ / ⌘- / ⌘0   zoom

## Files
⌘S save · ⌘Q quit · Esc cancel

mg / Emacs chords still work underneath — C-x C-s, C-n, M-w, and the rest — if that's how your hands already move.
