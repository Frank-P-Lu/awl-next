# Tables

A GFM table renders as an aligned pixel grid — cells placed by column, not by
space-padding (a proportional face can't align that way).

| Language | Paradigm      | Year |
|----------|:-------------:|-----:|
| Rust     | systems       | 2010 |
| Python   | multi         | 1991 |
| Lua      | scripting     | 1993 |

Prose after the table stays ordinary prose. A stray | pipe here is not a table.
