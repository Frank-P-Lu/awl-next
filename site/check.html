Check for updates

Version check.

awl never checks for updates on its own — no network request from the app, ever, at launch or on any timer. The "Check for Updates" command in the app records a local timestamp, then opens this page with the installed version as a URL parameter. This page's own script reads that parameter and compares it against a small version.json file this site regenerates on each deploy. The comparison happens here, in your browser, not in the app.
